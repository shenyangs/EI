# 超级管理员权限与后台规划

## 目标
把“超级管理员”从当前只管 AI 配置的半成品，升级成真正的系统管理后台。

## 超级管理员应负责的 7 个模块

### 1. 用户与身份
- 用户列表
- 用户详情
- 启用 / 禁用账号
- 重置登录状态
- 修改用户角色
- 查看所属机构与部门

### 2. 角色与权限
- 角色列表
- 角色权限摘要
- 角色权限编辑
- 新建自定义角色
- 用户组 / 角色组管理
- 查看谁拥有什么权限

### 3. AI 能力与模型
- 模型管理
- 模块分配
- 默认模型策略
- 失败回退策略
- 联网检索开关
- Prompt / 输出策略配置

### 4. 系统配置
- 站点级开关
- 注册策略
- 默认论文流程模板
- 默认会议规则模板
- Feature Flag

### 5. 数据治理
- 项目列表总览
- 项目删除 / 恢复
- 版本清理策略
- 参考文献数据校验
- 存储占用统计

### 6. 安全与审计
- 登录日志
- 超管操作日志
- 关键配置变更日志
- 权限变更日志
- 风险告警

### 7. 运行与诊断
- AI 模型连通状态
- 联网搜索状态
- 关键接口健康检查
- 当前环境配置摘要
- 错误与失败任务统计

## 建议的后台导航
- 总览仪表台
- 用户与身份
- 角色与权限
- AI 与检索
- 系统配置
- 数据治理
- 审计日志
- 运行诊断
