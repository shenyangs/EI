## 二、Prompt质量提升

### 旧Prompt（空洞、格式化）
```
你是一个面向服装、设计、时尚、人文社科与技术交叉研究的学术方向顾问。
基于用户提供的主题，生成3-5个具体的研究方向...
```

### 新Prompt（深度、专业、灵活）
```
你是一位资深的智能服饰研究专家，在可穿戴计算、电子纺织品、
健康监测、温控系统等领域有深厚的研究积累。

你的专业背景：
- 熟悉系统集成、传感器测试、穿戴实验、能耗分析等研究方法
- 长期关注IEEE Sensors、Advanced Materials Technologies等权威期刊
- 对柔性电子、能量收集、生理信号监测、个性化适配等前沿趋势有深入洞察

你的核心原则：
1. 基于真实学术脉络分析问题，不编造不存在的理论或文献
2. 提供具体、可操作的研究建议，避免空泛的通用表述
3. 结合领域最新趋势，指出真正的研究机会（research gap）
4. 评估研究可行性时考虑实际的数据、技术、时间约束
...
```

## 四、AI监督AI的质量评估体系

### 评估维度（加权）
1. **内容具体性**（25%）- 检测空洞词汇
2. **领域针对性**（25%）- AI评估
3. **学术规范性**（20%）- AI评估
4. **创新价值**（20%）- AI评估
5. **可行性**（10%）- AI评估

### 迭代机制
- 质量评分 < 7分 → 自动生成改进Prompt → 重新生成
- 最多迭代3次
- 每次迭代提高temperature以增加多样性

### 自我批评
- 对高质量内容（≥7分）生成自我批评
- 指出局限性和潜在风险
- 帮助用户建立批判性思维

## 五、研究方向生成改进

### 旧版本输出结构
```
1. 方向名称
2. 简要描述（1-2句话）
3. 研究价值
4. 可行性分析
```

### 新版本输出结构（8个维度）
```
1. 方向名称（15字以内，准确概括）
2. 研究背景与动机（当前领域问题+为什么值得研究）
3. 核心研究问题（1主问题+2-3子问题）
4. 研究创新点（与现有研究的差异+理论/实践价值）
5. 建议的研究方法（数据收集+分析方法+验证方式）
6. 可行性评估（数据可获取性+技术难度+时间成本+资源需求）
7. 潜在风险与应对
8. 相关研究脉络（继承/回应的研究+可借鉴的相邻领域）
```

## 六、论文评审改进

### 旧版本
```
1. 整体质量评分（1-10分）
2. 主要优点
3. 主要问题
4. 具体改进建议
5. 结构优化建议
```

### 新版本（5维度加权评分）
```
## 一、整体评价
- 质量评分：X.X/10
- 录用建议：直接录用/小修/大修/拒稿
- 一句话总结

## 二、详细评审维度
1. 研究问题与动机（权重20%）- 评分+问题+建议
2. 创新性与贡献（权重25%）- 评分+问题+建议
3. 研究方法与技术路线（权重20%）- 评分+问题+建议
4. 实验验证与结果（权重20%）- 评分+问题+建议
5. 写作质量与规范（权重15%）- 评分+问题+建议

## 三、主要优点（3-5点）
## 四、主要问题（致命/严重/一般）
## 五、具体修改建议（按章节）
## 六、给作者的建议
```

## 九、关键改进点总结

| 维度 | 改进前 | 改进后 |
|------|--------|--------|
| **Prompt深度** | 通用描述 | 领域专家角色+专业背景 |
| **内容结构** | 4个简单字段 | 8个深度维度 |
| **质量保障** | 无 | AI评估+迭代优化+自我批评 |
| **领域针对性** | 声明式 | 8个专业领域+研究方法+期刊+趋势 |
| **评审标准** | 笼统 | 5维度加权+具体评分标准 |
| **用户水平** | 无区分 | 4级用户（本科/硕士/博士/教授） |
| **灵活性** | 固定 | 深度/创新度可调 |
| **可追溯性** | 无 | 质量评分+迭代次数+生成时间 |
