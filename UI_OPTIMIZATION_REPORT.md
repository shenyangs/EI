## 三、缺陷清单与修复详情

### 3.1 布局与排版问题

#### 缺陷 #1: 论文类型选择器无法正常交互 ⚠️ P0
**问题描述：** 论文类型选择器点击后无法展开下拉菜单，用户无法选择论文类型  
**影响范围：** 首页论文类型选择功能  
**复现步骤：**
1. 访问首页
2. 点击"点击选择论文类型"区域
3. 下拉菜单不显示

**修复方案：**
- 重构组件结构，使用内联样式确保下拉菜单正确显示
- 移除可能导致冲突的 CSS 类
- 添加明确的点击事件处理

**修复状态：** ✅ 已完成  
**相关文件：** `/components/paper-type-selector.tsx`

#### 缺陷 #2: 错误消息显示样式不统一 ⚠️ P1
**问题描述：** 错误消息缺少统一的视觉样式和动画效果  
**影响范围：** 登录、注册、表单提交等所有错误提示场景  
**修复方案：**
- 创建统一的 `.error-message` 样式类
- 添加滑入动画效果
- 优化错误消息的视觉层次

**修复状态：** ✅ 已完成  
**相关文件：** `/app/globals.css`

## 四、优化方案实施详情

### 4.1 CSS 样式优化

#### 新增样式类
```css
/* 错误消息样式 */
.error-message {
  padding: 12px 16px;
  border-radius: var(--radius-sm);
  background: rgba(218, 90, 117, 0.1);
  border: 1px solid rgba(218, 90, 117, 0.2);
  color: var(--rose);
  font-size: var(--font-meta);
  margin-bottom: var(--space-md);
  display: flex;
  align-items: flex-start;
  gap: 8px;
  animation: slideDown 0.3s ease-out;
}

/* 移动端触摸优化 */
@media (max-width: 768px) {
  .primary-button,
  .secondary-button,
  .choice-card,
  .line-item--link,
  .project-card {
    -webkit-tap-highlight-color: transparent;
    touch-action: manipulation;
  }
}

/* 黑暗模式支持 */
[data-theme="dark"] {
  --bg: #0d1117;
  --text: #e6edf3;
  --muted: #8b949e;
  --accent: #58a6ff;
  /* ... 更多变量 */
}
```

#### 优化的过渡动画
```css
/* 统一的过渡效果 */
transition: all 200ms cubic-bezier(0.4, 0, 0.2, 1);

/* 主题切换动画 */
[data-theme] {
  transition: background-color 0.3s ease, color 0.3s ease;
}

/* 入场动画 */
@keyframes fadeIn {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```
